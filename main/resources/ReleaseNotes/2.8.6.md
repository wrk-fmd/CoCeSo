# Coordination Center Software - Release Notes

## Version 2.8.6

**Release date: 2025-07-??**

### Features
* **Enhanced Patient Registration Workflow**: Added dual-action save functionality in Patadmin
    - **Save**: Saves patient data and returns to registration overview
    - **Save & Add Next**: Saves patient data and immediately opens a new patient form for rapid batch processing
    - Improves efficiency during high-volume patient registration scenarios
    - Supports both German and English localization
